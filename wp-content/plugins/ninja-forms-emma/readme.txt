Emma's simple and stylish email marketing add-on for Ninja Forms.

== Screenshots ==

To see up to date screenshots, visit [NinjaForms.com](http://ninjaforms.com).

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `ninja-forms-emma` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the 'Forms -> Emma' menu item in your admin sidebar
4. Add your API Keys and Account ID
5. In the Ninja Forms Emails & Actions tab, you can now add a new action to subscribing a person to your Audience.
